## Cambios principales

- Se añade arquitectura dinámica por versión del mod.
- Se crea `data/mod_versions/catalog_ES.js` como índice de versiones.
- Se crea `data/mod_versions/3.0.0-rc.1/classes_ES.js` para clases.
- Se crea `data/mod_versions/3.0.0-rc.1/traits_ES.js` para rasgos, bonus y malus.
- Se crea `data/mod_versions/3.0.0-rc.1/categories_ES.js` para categorías humanas.
- Se crea `data/mod_versions/3.0.0-rc.1/recipes_summary_ES.js` para resumen de recetas.
- Se añade `rasgos.html` con todos los rasgos y efectos del mod.
- Se añade `clase.html?class=...` como ficha dinámica.
- Se añade `versiones.html` con estructura preparada para versiones futuras.

## Nota importante

Los PNG, Markdown y fichas HTML antiguas siguen incluidos como exportación estática. La parte mantenible es la web dinámica nueva.
