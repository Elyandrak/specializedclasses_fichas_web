# Specialized Classes fichas objetivas 2.4.0

## Cambios principales

- Añadido `data/mod_versions/3.0.0-rc.1/usage_ES.js` y `.json`.
- Nueva página `mecanicas.html`: objetos, herramientas, estaciones y uso real por clase.
- Cada ficha dinámica ahora muestra `Cómo se usa en partida` antes de rasgos/recetas.
- Corregida explicación del Archivista: en el mod 3.0.0-rc.1 la interacción verificada usa **palanca/crowbar**, no cincel+martillo.
- Añadidas aclaraciones para Relojero: **Tinkerer's Workstation** como base y **wrench/llave inglesa** en compatibilidad Millwright.
- Añadidas aclaraciones para Sastre: **spindle/huso**, **sewing kit** y **clothing assembly kit**; la aguja visual no se presenta como herramienta única principal.
- Añadidas notas de uso en rasgos concretos: `curator`, `archeologist`, `tinkerer`, `bloomer`, `clothier`, `clockmaker`, `vintner`.
